# Asthate Website

Static AM website with intake forms:
- `inquiry.html`
- `dd-request.html`
- `portal.html`

API backend for submissions:
- `server/index.js` (Express + MySQL)
- `server/schema.sql`

Deployment guide:
- `VULTR_DEPLOY.md`

Auto-deployment via GitHub Actions is enabled. Changes pushed to main will automatically deploy to the server.
